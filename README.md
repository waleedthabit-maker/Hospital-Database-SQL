# Hospital Database System

##  Overview
A relational database system designed using SQL to manage hospital operations including patients, doctors, appointments, and medical records.

---

##  Key Features
- Patient data management
- Doctor scheduling and appointments
- Medical test tracking and results
- Prescription and medication management
- Insurance integration
